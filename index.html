Proyecto Flutter

Tengo el placer de presentarte nuestro innovador proyecto implementado con Node.js para el backend, donde hemos creado una potente API para gestionar y medir el peso de las personas. Además, para la parte de frontend, hemos utilizado Flutter, permitiendo así crear una experiencia de usuario fluida y atractiva para nuestros usuarios.
En cuanto al Dashboard, hemos desarrollado una interfaz con tecnologías web como HTML y CSS, complementadas con funcionalidades dinámicas en PHP. Esto nos ha permitido ofrecer un Dashboard intuitivo y fácil de usar, donde los usuarios pueden acceder y visualizar sus datos de peso de manera clara y organizada.
